# Audit Initiatives Skill

## Context

Criteria for validating whether an Initiative is well-defined. Apply when auditing or drafting initiatives under a Key Result.

## Rules

1. Is tied to exactly one KR.
2. Has an expected impact on the KR metric.
3. Is larger than a task — may span multiple sprints.
4. Can be validated or invalidated with data.
5. Must be replaced if it does not move the KR.
