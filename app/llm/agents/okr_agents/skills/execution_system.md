# Execution System Skill

## Context

Defines the execution system used to structure and track work from strategy to daily action. Apply when displaying, validating, or auditing any level of the user's OKR hierarchy.

## Rules
1. **Objective** 
2. **Key Results** 
3. **Initiatives**
4. **Sprints**
5. **Tasks** 
6. Each level serves the one above it.
